TABLES CHEAT SHEET

Table Tags

Tag Name Description
<table> Table Contains tabular data within rows, colums, and cells.
<thead> Table Head Used to enclose heading elements like <th>.
<tbody> Table Body Should contain all of the table's data, excluding the table headings.
<tfoot> Table Footer Used to section off the bottom part of a table. Often used to contain sums, differences, and other data results.
<tr> Table Row Element used to add rows to a table.
<td> Table Data Used to add data to the table. *Adds columns to the table*.
<th> Table Heading Use this element to add titles to rows and columns.

Table Attributes

Attribute Name Values Description
scope Scope Row, col, rowgroup, colgroup This attribute allows you to position a heading either in a row or a column.
colspan Column Span Integer greater than or equal to 1 This attribute allows you to span columns across a web page.
rowspan Row Span Integer greater than or equal to 1 Similar to colspan this attribute allows you to span rows across a web page.
