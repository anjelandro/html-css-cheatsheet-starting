FORMS CHEAT SHEET

Forms Tags

Tag Name Description
<form> Form Element responsible for collecting information to send somehwere else.
<input> Input Creates an input field for a <form>.
<label> Label Used to explain to the user what an <input> is for.
<section> Section Used to represent a standalone section for which a more specific element cant be found. Usually has a heading as a child element.
<select> Select Used to create a dropdown list. Should have <option> element as a child.
<option> Option Used as a child element of <select> to create dropdown lists.
<datalist> Datalist Used with an <input type="text"> element. The <input> creates a text field that user can type into and filter options from the <datalist>.
<textarea> Text Area This element creates a bigger text field for user to write more text.

Forms Attributes

Attribute Name Values Description
type Type Text, Number, password, email, date, radio(multiple options, select one only), range (slider), color, checkbox(multiple options and choices), submit(creates a button) + This attribute determines how <input> renders on the web page and what kind of data can accept.
name Name No default value This is a very important attribute because without it the information in the <input> element wont be send when <form> is submitted.
id ID Matching value with for attribute for association purposes ID is used to associate elements like <input> and <datalist> to <label> element by matching the values of said elements.
for For Matching value with id and list attributes This attributes goes inside the <label> element and its used to associate <input>, <select>, and/or <datalist> with it.
step Step Equal or greater than 1 This attribute can be used along with type="number" attribute to create arrows inside the input field to increase or decrease by the value of the step attribute, it can also be used with type="range" to create a slider and the step attribute's value allows to control how smooth and fluid the slider works.
min and max Minimum and Maximum Whatever value we want to set as minimum and/or maximum (i.e. 1 and 100) Assigns a minimum and/or maximum attribute with a desired value for numeric inputs in forms.
minlength and maxlength Minimum length and Maximum lenght Whatever value we want to set as minimum and/or maximum (i.e. 3 and 200) Much like min and max attributes are used to set desired values for numeric inputs, minlength and maxlength attributes are used to set desired values for characters in text fields.
value Value Value for the input where the value attribute is nested Specifies the value of the <input> element.
list List Matching value with id attribute from a <datalist> element for association purposes Associates an <input> element with a <datalist> element.
required Required N/A We can use this attribute to make an input field mandatory in order to submit a form.
selected Selected N/A We can use this attribute inside of <option> element to make that element the default shown in the web page.
pattern Pattern Regular expression or "regex" We can use this attrribute for cases when we want user input to follow specific guidelines. If the input matches the regex, the form can be submitted.
